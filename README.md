# FATE_RPG_consultor
🎲 Fate Core Сonsultor: AI-Powered Game Master Assistant (LLM/RAG)
Интеллектуальный помощник для игроков и мастеров настольной ролевой игры (TTRPG) Fate Core. Система позволяет получать мгновенные и точные справки по правилам, используя современные методы обработки естественного языка (NLP).
🚀 Почему это важно?
В отличие от обычного поиска по ключевым словам, эта система понимает семантику правил. Она способна связать ваш игровой контекст с конкретными механиками системы Fate, помогая не прерывать игру на долгие поиски в PDF-файле.
🛠 Технологический стек
Проект реализован на базе архитектуры RAG (Retrieval-Augmented Generation):
LLM: Qwen 2.5 (1.5B) Instruct — компактная, но мощная модель, оптимизированная для следования инструкциям.
Embeddings: sentence-transformers/all-MiniLM-L6-v2 — легковесная и быстрая модель для векторного представления текста.
Vector Search: Кастомная реализация семантического поиска на базе NumPy (Cosine Similarity) для обеспечения максимальной производительности в облачных средах с ограниченными ресурсами (CPU).
Интерфейс: Gradio — для быстрого развертывания веб-приложения.
Deployment: Hugging Face Spaces.
📖 Методология
Preprocessing: Правила в формате Markdown были очищены и разбиты на семантические блоки (chunks) с учетом иерархии заголовков.
Indexing: Каждый блок векторизован и помещен в индекс для быстрого поиска.
RAG Pipeline: При получении вопроса система находит 3 наиболее релевантных фрагмента правил и подает их в LLM в качестве контекста.
Prompt Engineering: Использован специализированный системный промпт, минимизирующий «галлюцинации» модели и заставляющий её опираться только на официальный текст правил.
📈 Результаты тестирования
Система успешно прошла стресс-тесты на знание базовых механик:
Фактология: Точное перечисление четырех типов действий.
Нюансы: Объяснение разницы между "Success at a Cost" и провалом броска.
Логика: Интерпретация игровых ситуаций (например, использование Аспектов).
